Project information

  • Title: BreezoScan
  • Description: This website uses Global Air Quality data to provide insights on pollution levels and predicts the Air Quality Index (AQI) based on contaminant information
  • Github URL: https://github.com/sahana-manju/Breezoscan_User
  • App Link: https://cmsefinal-jmfv3xchfgdeabujtoywg3.streamlit.app/

Details

  • Methodologies: This project uses Machine Learning models like Decision Trees, XGBoost, Random Forest, and SVC. Built with Streamlit, Seaborn, Plotly, and Matplotlib for interactive visualizations
  • Results & Impact: SVC achieved best results compared to other models with an Accuracy of 97% and F1 Score of 96%
