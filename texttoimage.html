Project information

  • Title: FloralVisionary
  • Description: This project converts text descriptions of flowers into images, bringing floral details to image based on given descriptions
  • Github URL: https://github.com/sahana-manju/Text_to_Image_Flowers

Details

  • Methodologies: This project used 3 different approaches: XLNET and GAN, GRU and GAN for text encoding and image generation respectively, along with Stable Diffusion model finetuning
  • Results & Impact: The results show that Stable Diffusion achieved the best clarity with a score of 737.77 (Mean: 1.37, Std: 0.21), outperforming GAN and XLNet, which had an FID score of 1355.40 (Mean: 1.0, Std: 0.0). This superior performance in image quality and detail was also confirmed in human evaluations
